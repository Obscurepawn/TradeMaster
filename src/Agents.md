# TradeMaster Root Context

## Overview
TradeMaster is a CLI-based backtesting framework for quantitative trading strategies.

## Modules
- `config`: Configuration management.
- `data_loader`: Market data acquisition and caching.
- `strategy`: Trading logic.
- `backtest`: Simulation engine.
- `proxy`: Network proxy management.
- `drawing`: Visualization.

## Scripts
- `run_backtest.sh`: Execute a strategy.
- `run_tests.sh`: Run the test suite.
